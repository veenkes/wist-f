# Чеклист тестирования Notifications

## 1. Страница Notifications
- [ ] Страница загружается без ошибок
- [ ] Статистика отображается (Total, Sent, Failed, Pending)
- [ ] Список уведомлений загружается
- [ ] Фильтры работают (по типу, по статусу)
- [ ] Поиск работает
- [ ] Кнопка "Create Notification" открывает модалку

## 2. Создание уведомления (CreateNotificationModal)
- [ ] Модалка открывается
- [ ] Все поля формы отображаются
- [ ] Выбор типа уведомления работает
- [ ] Ввод заголовка работает
- [ ] Ввод сообщения работает
- [ ] Выбор получателей работает:
  - [ ] All Users
  - [ ] All Students
  - [ ] All Parents
  - [ ] All Staff
  - [ ] Custom Selection - открывает список студентов
- [ ] Custom Selection:
  - [ ] Список студентов загружается
  - [ ] Поиск студентов работает
  - [ ] Выбор студентов работает (чекбоксы)
  - [ ] Нет бесконечного цикла при выборе
- [ ] Выбор методов доставки работает (in-app, email, telegram)
- [ ] Загрузка файлов:
  - [ ] Click to upload работает
  - [ ] Drag & drop работает
  - [ ] Загруженные файлы отображаются
  - [ ] Удаление файлов работает
- [ ] Отправка уведомления работает
- [ ] После отправки форма очищается
- [ ] После отправки список обновляется

## 3. Просмотр деталей уведомления
- [ ] Кнопка "View Details" открывает модалку
- [ ] Все данные отображаются корректно
- [ ] Модалка закрывается

## 4. WebSocket (если подключен)
- [ ] Счетчик непрочитанных обновляется в реальном времени
- [ ] Нет ошибок в консоли

## 5. Ошибки и edge cases
- [ ] Обработка ошибок API (нет сети, 500 и т.д.)
- [ ] Валидация формы (пустые поля)
- [ ] Обработка пустых списков
- [ ] Обработка больших списков студентов
